The Problem

The founder was spending money on 4 platforms with no way to know which was converting to paid subscribers. Every channel looked equally active. None of the data talked to each other.

What I Built
01
Data Unification
Pulled API data from all 4 platforms into a single Excel model with consistent UTM logic.
02
CAC Model
Calculated blended and per-channel CAC. Surfaced Instagram as 2.7× more efficient than Facebook.
03
Power BI Dash
Automated weekly refresh dashboard the founder could update herself without a data analyst.
04
Recommendation
Reallocated 60% of Facebook budget to Instagram + YouTube. Blended CAC dropped 34%.
